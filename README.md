# Bookstore Application 📚

A simple frontend-only Bookstore Application built with React. The app currently includes login, signup, categories, and displays books in a card format. Future enhancements will add features like cart functionality and admin controls.

## Features ✨

- **Login**: Users can log in to access their account 🔐
- **Signup**: New users can create an account 📝
- **Categories**: Browse books by different categories 📑
- **Book Cards**: View book details in interactive cards 🃏

## Technologies Used ⚙️

- **Frontend**: React, React Router, React Hooks
- **Styling**: CSS, Bootstrap (or any other CSS framework you're using)
